# VanillaShader

VanillaShader é um shader pack original para Minecraft Java desenvolvido para Iris Shaders.

## Objetivos

- Alto desempenho
- Aparência Vanilla aprimorada
- Compatível com Iris
- Otimizado para GPUs integradas AMD
- Perfis:
  - Leve
  - Médio
  - Ultra
  - Absolute Cinema

## Hardware alvo

- Ryzen 5 7000 Series
- Radeon Graphics integrada
- 8 GB RAM
- SSD
- Windows 11
